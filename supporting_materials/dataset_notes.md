# Dataset Notes

The included dataset is prepared for practical demonstration of the CORTEX MVP.

It includes:

- Multiple user IDs.
- Time-series dates.
- Wearable-style input features.
- A target recovery score.
- A target health-state class.

The app does not require target columns for live analysis. It only requires the base input features. The target columns are included so the training/evaluation script can demonstrate train/test behavior.

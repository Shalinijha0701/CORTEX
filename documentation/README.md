# CORTEX Prototype Submission

## What This Package Contains

- `source_code/` - Streamlit app, CORTEX health intelligence engine, dataset, requirements, and test scripts.
- `documentation/` - Technical documentation and architecture diagram.
- `presentation/` - Project presentation deck.
- `demo_video/` - Demo video plan and submission link placeholder.
- `supporting_materials/` - Prototype mockups and supporting screenshots.

## How To Run The Prototype

1. Open a terminal in `source_code/`.
2. Create and activate a Python virtual environment.
3. Install dependencies with `pip install -r requirements.txt`.
4. Run `streamlit run app.py`.
5. Upload a compatible CSV or use the included `sample_health_data.csv`.

## MVP Flow

User health CSV -> visible upload validation -> cleaning -> personal baseline creation -> CORTEX Health Intelligence Engine -> dashboard, intelligence hub, patient alerts, clinician portal, anomalies, recovery score, stress trend, health state, PDF report, and CSV exports.

## Important Note

CORTEX is a wellness intelligence prototype. It is not a clinical diagnosis product and should not be used as a replacement for medical advice.

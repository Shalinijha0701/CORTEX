# CORTEX Demo Video Script

## 0:00 - 0:20 Introduction

Hello, this is CORTEX, a personalized health intelligence prototype. It converts wearable-health CSV data into recovery, stress, sleep, anomaly, and health-state insights.

## 0:20 - 0:45 Problem

Wearables collect a lot of data, but many dashboards only show generic charts. CORTEX compares a user's current signals against their own baseline, which makes insights more personalized.

## 0:45 - 1:20 Upload And Analyze

Open the Streamlit app, load the included sample data, and select a user profile. The app validates the CSV, normalizes stress values, cleans missing values, builds the baseline, and runs the CORTEX Health Intelligence Engine.

## 1:20 - 2:00 Dashboard

Show the recovery score gauge, stress index, sleep quality, HRV delta, resting heart rate, and health state confidence. Then show the HRV-versus-stress chart and recovery trend.

## 2:00 - 2:30 Insights

Open Data Upload Validation and show each validation step. Then open the Intelligence Hub and show anomaly alerts, recommendations, baseline comparison, model metrics, and the PDF report export.

## 2:30 - 3:00 Architecture And Close

Open Profile & Baseline, Patient Alerts, and Clinician Portal. Show the multi-patient queue and selected-patient PDF export. Mention future scope: wearable API integration, authentication, secure storage, and doctor dashboard. Close by saying the MVP is a wellness prototype and not a medical diagnosis tool.
